Number representation in Verilog:

Declaration:
<size> <radix> <value> =

<size> <- No.of Bits

<radix> <- Number Representation

<value> <- Consecutive value - O to f, x, z

  • Binary -> b (or) B
  • Octal -> o (or) O
  • Decimal -> d (or) D
  • HexaDecimal -> h (or) H

Ex1: 4'b 0101

Ex2: 8'h 0101xxxx

Ex3: 12'o 3zx7 = 011 zzz xxx 111
